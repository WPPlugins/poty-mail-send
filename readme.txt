Plugin to send mails from the administration of wordpress.

Plugin para poder mandar mails desde la administración de wordpress.

== Description ==

Send mail easily from wordpress administration.
Load the list of wordpress users with their email addresses to send an email, through a simple form.

Envia mails facilmente, desde la administraci&oacute;n de wordpress.
Carga la lista de usuarios de wordpress con sus direcciones de correo para poder mandarles un correo, a trav&eacute;s de un sencillo formulario. 



== Installation ==

1. Unzip the file into the plugins folder of your wordpress '/ wp-content/plugins'.

2. Activate it from the plugins section of your administration.



1. Descomprime el archivo dentro de la carpeta plugins de tu wordpress '/wp-content/plugins'.

2. Activalo desde la secci&oacute;n plugins de tu administraci&oacute;n.
